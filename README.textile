h1. Twitter Node.js Socket.IO Example

Quick experiment of direct streaming from twitter to the browser, with no polling, using Node.js and Socket.IO.
(Based on https://github.com/andregoncalves/twitter-nodejs-websocket, which is in turn a follow up to Ruben Fonseca "version":http://github.com/rubenfonseca/twitter-amqp-websocket-example with Ruby and AMQP).

h2. Requirements:

* Node.js.
* HTML5 WebSockets capable browser.

h2. How To:

* Download and install node.js (instructions "here":http://nodejs.org/#download).
* Download the project from "github":http://github.com/threetee/twitter-nodejs-socketio.
* Start the server: @node server.js <twitter_username> <twitter_password> <optional_keyword>@.
* Open @index.html@ with a WebSocket compatible browser (Chrome or Webkit nightly).

h2. Libraries used:

* "Minimal websocket library":http://github.com/ncr/node.ws.js for node.js from Jacek Becela.
